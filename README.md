# Portfolio Website

My professional portfolio showcasing projects and skills.

## Deployment Process
1. Initialized git repository.
2. Pushed code to GitHub.
3. Connected GitHub repository with Netlify for continuous deployment.
4. Set build command to `npm run build` and output directory to `dist`.

## Challenges Faced & Solutions
- **Challenge:** Layout was appearing broken and stretched across the screen.
- **Solution:** Implemented a global `container` class with `max-width: 1100px` to maintain visual hierarchy and center the content.
- **Challenge:** Smooth navigation in a Single Page Application.
- **Solution:** Used CSS `scroll-behavior: smooth` with anchor IDs for seamless section navigation.
